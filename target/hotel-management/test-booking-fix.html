预订功能修复验证

🐛 原始问题：

错误： java.lang.IllegalArgumentException: 入住人数必须为正数

🔍 根本原因：

BookingController.saveBooking() 方法中没有处理 guestsCount 参数

✅ 修复方案：
  1. 添加 guestsCount 参数获取
  2. 增加参数验证（不为空且为正数）
  3. 设置 booking.setGuestsCount(guestsCount)
  4. 统一错误信息为 errorMessage
📋 测试步骤：
  1. 访问：http://localhost:8080/hotel-management/admin/booking/add
  2. 填写所有必填信息
  3. 提交表单
  4. 应该成功创建预订
